# Animalhospital - 도란도란 

# 1. 주제
*동물병원 정보찾기

# 2. 프로젝트목적
*  집근처 가장 가까운 동물병원을 보다 간편하게 찾을 수 있도록 한다.

 # 4. 역할분담
* 신덕용: 매장, 동물병원 위치 데이터 출력, 동물병원 리뷰 크롤링
* 김명호: 회원등록, 회원탈퇴, 스프링 시큐리티
* 박준희: 리뷰작성, 리뷰삭제, 리뷰수정
* 유희진: 매장찾기, 동물병원 위치 데이터 출력
 
 
 # 4. 개발우선순위
* 1순위. 매장찾기, 리뷰작성
* 2순위. 로그인
* 3순위. 게시판, 고객센터
